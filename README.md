# Namma HomeStay

Namma HomeStay is a lightweight mobile-responsive hospitality management application developed using Firebase and Generative AI integration.

The application helps rural home-stay owners manage:
- Daily food menus
- Tourist place recommendations
- Traveler inquiries
- Hospitality information

## Features

- Firebase Authentication
- Firestore Cloud Database
- AI-generated food descriptions
- Inquiry management system
- Local guide management
- Responsive mobile-friendly UI

## Technologies Used

- HTML
- CSS
- JavaScript
- Firebase Authentication
- Firebase Firestore
- Google Gemini AI
- VS Code

## AI Integration

Google Gemini AI was integrated to generate attractive hospitality-style food descriptions dynamically.

## Deployment

The project is deployed as a lightweight responsive web application for better accessibility and reduced system resource requirements.
